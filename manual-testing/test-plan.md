# Test Plan - Book Store Application

## 1. Objective 
To verify the funcstionality, usability and reliability of the Book Store web application.

## 2. Scope
### In Scope
- User Registration and Login
- Browse Books
- Add to cart
- Place Order
- Logout

### Out of Scope
- Performance testing

## 3. Test types
- Functional Testing
- Regression Testing
- Smoke Testing

## 4. Test Environment
- OS: Windows
- Browser: Chrome
- Application URL: http://localhost:5173

## 5. Tools
- Manual Testing: Test cases document
- Automation: Cypress (Javascript)
- API Testing: Postman
- Defect Tracking: Markdown document

## 6. Entry & Exit Criteria
### Entry
- Application build is available
- Test data created

### Exit
- All critical test cases passed
- No open high-severity defects
